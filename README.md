A simple React-based password generator that helps you to generate passwords with custom length and options.

# Features:
- Generate random password.
- Set custom length to password upto 20 letters.
- Include numbers & characters.
- Copy password to clipboard.

# Installation:
```bash

# Clone the repositary
git clone https://github.com/subodh-baniya/Password-generator.git

# Go to the project folder
cd password-generator

# Install dependencies
npm install

# Start the application
npm run dev
```
